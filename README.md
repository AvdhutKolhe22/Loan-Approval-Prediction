# Loan Approval Prediction using Machine Learning

## Overview
This project predicts whether a loan application will be approved or rejected using machine learning algorithms.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Models Used
- Logistic Regression
- Decision Tree
- KNN

## Results
- Decision Tree achieved highest accuracy (~97%)
- Logistic Regression and KNN achieved ~90%

## Features
- Data preprocessing and cleaning
- Handling missing values
- Feature scaling
- Model comparison
- Visualization of results
